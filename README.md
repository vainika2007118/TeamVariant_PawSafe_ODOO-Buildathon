# 🐾 PawSafe - Stray Animal Rescue ERP

**Team Variant** | ODOO Buildathon | ACM MPCE Chapter x Microsoft Tech Club

## Theme
Public & Institutional ERP

## About
PawSafe is a complete ERP system built on Odoo for managing the entire lifecycle of a stray animal — from rescue to adoption.

## The Problem
India has over 30 million stray animals. NGOs trying to help them are working with WhatsApp groups, paper registers and Excel sheets. Rescues get delayed. Medical records get lost. Bad adopters get approved again. PawSafe solves this.

## Features
- 🐕 Animal profiles with photo, personality and medical history
- 🏠 Shelter management with real-time capacity tracking
- 🚨 Rescue case management with priority levels
- 🩺 Complete veterinary records
- 🏡 Foster care coordination
- 💰 Sponsor tracking per animal
- ✅ Adoption screening with automatic blacklist protection
- 📊 Kanban view showing animal journey from stray to adopted
